Face Recognition Attendance System

Features

Built for easy demos and teacher reviews — our system provides a clean frontend experience with practical, demo-ready features:

  • Simple registration: Upload a photo or register via webcam in seconds.
  • Fast verification: Use the webcam to match a captured frame against registered references — instant feedback.
  • Attendance history: Full running log that appends each successful verification (keeps teacher records for the demo).
  • Privacy-first demo: Data is stored locally in your browser (no server), perfect for classroom demonstrations and assessments.
  • Export & share: Download attendance history as CSV for grading or records.
  • Manage students: Delete reference photos anytime — simple maintenance UI.
